# Руководство пользователя — Аудио Рекордер

## Что делает программа

Записывает звук с микрофона и системного звука одновременно, автоматически
разбивает запись на отдельные файлы по паузам в речи.

**Типичный сценарий:** конференция в Zoom — голос участников пишется с системного
звука (loopback), твой голос — с микрофона. Файлы отдаются в Whisper для транскрипции.

---

## Запуск

```
python recorder_ui.py
```

## Описание всех настроек

### Папка сохранения
Куда сохраняются файлы `recording_YYYY-MM-DD_HH-MM-SS.mp3/.wav`.
По умолчанию: папка `recordings/` рядом с программой.

---

### Порог тишины (RMS)
**Диапазон:** 50 — 5000 | **По умолчанию:** 500

Уровень громкости ниже которого считается тишиной. RMS — среднеквадратичная амплитуда.

| Значение | Поведение |
|---|---|
| 200–400 | Очень чувствительный — реагирует на тихий шёпот и фоновый шум |
| 500–800 | Нормальный — рекомендуется для обычной речи |
| 1000–2000 | Менее чувствительный — только громкая речь |

Смотри на VU-метры — белая черта показывает текущий порог.
Изменяется «живым» во время записи.

---

### Длительность паузы
**Диапазон:** 0.3 — 10.0 сек | **По умолчанию:** 0.9 сек

Сколько секунд тишины нужно чтобы закрыть текущий файл и начать ждать следующую речь.

---

### Мин. длит. речи
**По умолчанию:** 0.5 сек | Чекбокс: снять = отключить фильтр

Файлы короче этого значения не сохраняются — отсекает случайные шумы (кашель, стук).

---

### Мин. время записи
**По умолчанию:** 0.0 (выкл)

Пока не прошло N минут с начала сегмента — тишина игнорируется, файл не закрывается.
Пример: `5.0 мин` — каждый файл будет не короче 5 минут.

---

### Время откл. при тишине
**По умолчанию:** 0.0 (выкл)

Если нет звуков дольше N минут — запись останавливается автоматически.
Защита от «забыл остановить».

---

### Формат файла
**MP3** (~1 МБ/мин при 128 кбит/с) или **WAV** (~11 МБ/мин, без сжатия).
WAV лучше для точной транскрипции через Whisper.

---

### Битрейт MP3
64 / 96 / **128** / 192 / 320 кбит/с.

| Битрейт | Размер ~1 мин | Для чего |
|---|---|---|
| 64 кбит/с | ~0.5 МБ | Только речь, хватает для Whisper |
| 128 кбит/с | ~1.0 МБ | Хороший баланс — рекомендуется |
| 192+ кбит/с | ~1.5 МБ+ | Если важно высокое качество |

---

### Непрерывная запись
Параллельно пишется один сплошной файл `full_*.mp3` от «Начать» до «Остановить».
Страховка: даже если разбивка что-то пропустила, полная запись есть.

**Папка для full_\*:** отдельная папка. Пусто = та же что и основная.

---

### Микрофон
Выбор устройства записи. **Авто** — микрофон по умолчанию (WASAPI).
Кнопка **↻** — обновить список после подключения нового оборудования.

---

### Системный звук
Выбор loopback-устройства (захват того что играет в динамиках).
**Авто** — loopback дефолтного динамика.

Примеры:
- `Realtek(R) Audio [Loopback]` — весь системный звук
- `Авто` — то же самое, автоматически

---

## VU-метры

- **Зелёный** — ниже порога тишины
- **Жёлтый** — 65–100% от порога
- **Красный** — выше порога (речь записывается)
- **Белая черта** — текущий порог

**Верхний предел шкалы** — максимум на шкале (по умолч. 2000).

---

## Панель статуса

- **Серый кружок** — ожидание речи
- **Красный кружок (мигает) + «Запись...»** — идёт запись сегмента
- **Красный таймер** — время текущего сегмента
- **Серый «Общее: M:SS»** — полное время с нажатия «Начать запись»

## Список записанных файлов

Двойной клик — открыть в плеере. Кнопка **Открыть папку** — открыть в Проводнике.

## Как работает детектор голоса

```
ОЖИДАНИЕ ──► РЕЧЬ ──► ТИШИНА ──► ОЖИДАНИЕ (новый файл)
               ▲          │
               └──────────┘  (звук вернулся до конца паузы)
```

**ОЖИДАНИЕ:** ведём скользящий буфер 0.25 сек (пре-ролл). Как только уровень
превысил порог — переходим в РЕЧЬ, добавляем пре-ролл в начало файла.

**РЕЧЬ:** пишем фреймы. Уровень упал — переходим в ТИШИНА.

**ТИШИНА:** продолжаем писать. Звук вернулся — обратно в РЕЧЬ. Тишина дольше
«Длительности паузы» — сохраняем файл с пост-роллом 0.35 сек.

## Рекомендуемые настройки

### Zoom-конференция
```
Порог тишины:       500–700
Длительность паузы: 1.5 сек
Мин. длит. речи:    0.5 сек (вкл)
Формат:             WAV или MP3 128 кбит/с
Непрер. запись:     вкл
Системный звук:     Авто
```

### Запись лекции / монолога
```
Порог тишины:       400–600
Длительность паузы: 3.0 сек
Мин. длит. речи:    1.0 сек
Мин. время записи:  2.0 мин
Формат:             WAV
```

### Длинное совещание без присмотра
```
Длительность паузы:      2.0 сек
Время откл. при тишине:  30 мин
Непрер. запись:          вкл
```
