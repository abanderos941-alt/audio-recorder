# Установка — Аудио Рекордер

## Требования

- Windows 10 / 11 (x64)
- Python 3.10 или новее

---

## 1. Установка зависимостей

Открой PowerShell или командную строку в папке проекта:

```
pip install -r requirements.txt
```

| Пакет | Для чего |
|---|---|
| `pyaudiowpatch` | Захват звука (WASAPI loopback), только Windows |
| `lameenc` | MP3-кодирование |
| `numpy` | Обработка аудио: микширование, вычисление RMS |

Tkinter входит в стандартную поставку Python — ничего дополнительного не нужно.

---

## 2. Запуск

```
python recorder_ui.py
```

Настройки сохраняются автоматически в `recorder_settings.json`.

---

## 3. Удалённое использование через Parsec / AnyDesk / TeamViewer

Для записи Zoom-конференций удалённо используй **Parsec, AnyDesk или TeamViewer** —
они подключаются к консольной сессии где доступны все физические аудиоустройства.

**Почему не RDP:** Windows RDP создаёт изолированную сессию с виртуальным
аудиоустройством. Физические устройства (Realtek, микрофон) из RDP-сессии недоступны.

### Рабочий процесс

```
1. Подключись через Parsec/AnyDesk
2. Запусти Zoom → подключись к конференции
3. Запусти recorder_ui.py → нажми "Начать запись"
4. Отключись от Parsec/AnyDesk
   └── recorder_ui.py продолжает работать в консольной сессии
5. Подключись снова когда нужно остановить запись
```

---

## 4. Проверка аудиоустройств

```
python list_devices.py
```

Показывает все устройства с индексами — используй для диагностики или выбора
конкретного устройства в настройках.
